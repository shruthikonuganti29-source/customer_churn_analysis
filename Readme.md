# 📊 Customer Churn Analysis Dashboard

This project analyzes telecom customer data to identify churn patterns, customer behavior, and revenue trends using **SQL, Python, and Power BI**.

---

## 🚀 Project Objective

The goal of this project is to help businesses understand:

* Why customers leave (churn)
* Which contract types retain customers better
* Revenue patterns across customer groups
* Preferred payment methods
* Monthly charge trends

---

## 🛠️ Tools & Technologies Used

* **Python** (Pandas, Matplotlib)
* **SQL / SQLite**
* **Power BI**
* **Jupyter Notebook**
* **VS Code**

---

## 📂 Dataset

Dataset used: IBM Telco Customer Churn Dataset

Includes customer details such as:

* Gender
* Tenure
* Contract Type
* Internet Service
* Payment Method
* Monthly Charges
* Total Charges
* Churn Status

---

## 📊 Dashboard Features

### KPI Cards

* Total Customers
* Monthly Revenue
* Churn Customers

### Visualizations

* Customers by Contract Type
* Churn by Internet Service
* Customers by Payment Method
* Average Monthly Charges by Contract

---

## 🔍 Key Business Insights

* Month-to-month contracts show higher churn rates.
* Fiber optic customers contribute the highest churn count.
* Electronic check is the most common payment method.
* Two-year contracts have lower monthly charges and better retention.
* Customers on long-term contracts are more stable.
